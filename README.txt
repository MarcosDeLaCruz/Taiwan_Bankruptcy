# Bankruptcy Prediction in Taiwan 🇹🇼

This project uses financial indicators to predict company bankruptcy using a machine learning pipeline with Random Forests and oversampling techniques.

## 🔍 Project Overview

- Load and clean data from `data.csv`
- Explore class imbalance and apply `RandomOverSampler`
- Train a Random Forest model with `GridSearchCV`
- Evaluate performance with accuracy, confusion matrix, and classification report
- Save and reuse the model for future predictions
